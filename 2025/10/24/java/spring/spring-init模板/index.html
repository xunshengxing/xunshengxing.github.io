spring-init模板

发表于2025-10-24|更新于2025-10-24|spring
|总字数:4.8k|阅读时长:24分钟|浏览量:

简介

基于springboot的项目初始模板，用于快速整合后端

目前实现：

  1. 全局登录校验拦截器JwtTokenUserInterceptor
  2. 全局异常处理器GlobleException
  3. 自定义错误码ErrorCode
  4. Knife4j接口
  5. 全局跨域处理CorsConfig
  6. 封装通用响应类Result
  7. logback日志按天滚动logback-spring.xml
  8. 基于MockMVC的单元测试

计划实现：

  1. springbootAOP切面编程
  2. Spring事务注解
  3. Spring Scheduler 定时任务
  4. 集成Redis、OSS对象存储
  5. 业务代码生成器（自动生成增删改查）
  6. Spring Session Redis 分布式登录
  7. 自定义权限注解 + 全局校验
  8. 文件上传
  9. 集成ES搜索引擎

模板版本：

  • springboot2.7.16

  • jdk8

Knife4j

是swagger一个Restful Api开源框架，OpenApi规范的来源)的增强版，最初是美化Swagger的界面，后面升级为Knife4j

参考文档：快速开始 | Knife4j

适用于spring2.4-3.0

直接使用yml文件配置无需编写配置类

Jwt

认识

一种非加密，防止篡改的认证，无需存储在服务器

三个组成部分，用.分隔：

  • Header 标头（存放令牌类型与签名算法）

  • Payload 有效载荷（存放token信息）

  • Signature 签名 (防止篡改，唯一加密的地方)

    下面显示了一个 JWT，它对前面的标头和有效负载进行了编码，并使用秘码进行了签名.
