用 Java 语言实现，允许用户使用简单的编程模型实现分布式处理

核心组件

  • HDFS 分布式文件存储系统 ：解决海量数据存储
  • YARN 集群资源管理和任务调度框架：解决资源任务调度，支持各种计算引擎运行
  • MapReduce 分布式计算框架：解决海量数据计算

特性优点

通用性：与具体的业务脱钩

简单：编程模型使用简单

架构变化

Hadoop 集群

  1. 包括两个集群：HDFS 集群、YARN 集群
  2. 两个集群 逻辑上分离，物理上在一起
  3. 两个集群都是标准的主从架构集群

MapReduce 是计算框架、代码层面的组件，没有集群之说
